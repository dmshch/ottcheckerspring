# OttChecker

In plans - the simple application for visualisation of chunk rotation and OTT app.protocols errors - HTTP Live Streaming (HLS), Dynamic Adaptive Streaming over HTTP (MPEG-DASH) and Smooth Streaming (SS).

1. HLS - Only for Live Media Playlists - access for Master Playlists, Media Playlists, Media Segments (not according to the standard)

2. MPEG-DASH - in progress (not according to the standard).

3. SS - Not yet.
